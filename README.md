# Node.js + NestJS + hexagonal architecture practice

[Lessons](https://www.youtube.com/watch?v=vX5keKH6iZU&pbjreload=101)

## Installation

```bash
$ yarn
```

## Running the app

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev

# production mode
$ yarn start:prod
```

## Test

```bash
# unit tests
$ yarn test
```
